AI Printer — Device Protocol Demo

Speaks the Cheeko device protocol on /ws: hello handshake → line_art_* → renders the printed 1-bit bitmap.
ⓘ Browsers can't produce the bare 16 kHz Opus frames the real device sends, so this demo drives generation with a text prompt (it opens a short browser-protocol connection for that). It still performs the real device hello handshake and renders the line_art output exactly as the device's printer would. For the real audio path, use ai_printer_client.py (Python, raw Opus).
